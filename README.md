# AI Chatbot 🤖

A simple AI chatbot application built using HTML, CSS, and JavaScript.

This chatbot provides an interactive chat interface where users can send messages and receive automated responses.

## Features
1. Interactive chatbot UI
2. User and bot chat messages
3. Auto-generated responses
4. Responsive design
5. Enter key support
6. Smooth chat scrolling

## Technologies Used
- HTML
- CSS
- JavaScript
